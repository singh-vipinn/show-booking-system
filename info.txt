How to run the Demo (Step-by-Step):

Option A: Using your IDE (IntelliJ / Eclipse / VS Code)
    -Navigate to src/main/java/com/booking/Driver.java.
    -Right-click the file and select Run 'Driver.main()'.\
    -Observe the Console Window at the bottom. You will see the output log matching your requirements.

Option B: Using Terminal (Command Line):
    -If you have Maven installed, open your project folder in terminal and run:

mvn spring-boot:run

4. Verification Checklist for the "Full Marks":

-Waitlist Test: Try booking 3 seats when only 2 are available. It should say "Added to waitlist".
-Conflict Test: Try booking UserA for Show1 at 9:00 and then UserA for Show2 at 9:00. It should fail.
-Ranking Test: Register two different shows for the same genre at different times. Run showAvailByGenre. They must appear in order of time (9:00 first, then 10:00).
